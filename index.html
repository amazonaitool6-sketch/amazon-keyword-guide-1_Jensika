Step-by-Step Research Process

Identify Your Top Keyword

This is your primary keyword phrase (not single word)

Example Formula:
[Product Type] + [Main Feature]

e.g., "fish oil for dogs"

Find Top Competitors

  • Search your main keyword on Amazon
  • Select 3-10 top organic products (skip sponsored)
  • Choose products with highest revenue
  • Match your price point and design
  • Use Xray to filter and extract ASINs

Extract ASINs & Run Cerebro

Input all competitor ASINs into Cerebro to discover 4,000+ keywords

‚ö†Ô∏è CRITICAL FOR NEW LAUNCHES:

Target keywords with CPR (Cerebro Product Rank) under 50!

  • CPR 0-50: Easy to rank (LOW competition) ‚úì
  • CPR 51-100: Medium difficulty
  • CPR 100+: Hard to rank (HIGH competition)

Keyword Types & Strategy

Short-tail Keywords
1-2 Words

High volume, high competition
Example: "Pendant Light"

Long-tail Keywords
3+ Words

Lower competition, higher conversion
Example: "E27 Adjustable Pendant Light for Kitchen"

Match Types Strategy

Match Type How It Works Best For Word Count
Exact Match Shows only for exact keyword phrase Long-tail keywords 4+ words
Phrase Match Keywords together in order, extras allowed Multi-word phrases where order matters 2-3 words
Broad Match Keywords in any order with other words Generic 1-4 word keywords 1-4 words

Essential Keyword Metrics

Search Volume

Exact/Long-tail: 1-5,000
Broad match: Higher volumes OK

Relevance

Must directly relate to your product

Competition

Too high = hard to rank

Position Rank

Set to 1-50 (top competitors only)

  • Remove ALL branded keywords from competitors
  • Filter out irrelevant search terms
  • Check for misspellings of competitor brands
  • Use "Phrases Containing" to exclude brand names

üì± MOBILE CRITICAL:

Title Optimization

Title Formula:
Brand + Product Type + Key Feature + Size/Color/Quantity
Characters 1-80
MOBILE
  • ‚úì Brand name
  • ‚úì Main product keyword
  • ‚úì Top feature/benefit
  • ‚úì Key specification
Characters 81-200
DESKTOP
  • ‚úì Additional features
  • ‚úì Use case
  • ‚úì Size/color/specs
  • ‚úì Secondary keywords

‚úì DO:

  • Capitalize First Letter Of Each Word
  • Use numerals (1.2m instead of 1.2 meters)
  • Include brand at start
  • Front-load high volume keywords
  • Use natural, readable language

‚úó DON'T:

  • Use ALL CAPS (except E27, LED)
  • Add 'Best', 'Sale', '#1', 'Hot'
  • Use special characters: !@#$%^&*
  • Repeat keywords unnecessarily
  • Exceed 200 characters
Example Perfect Title:
E27 Pendant Light - Adjustable Hanging Ceiling Lamp for Kitchen, Dining, Bar (Yellow Brass)

Bullet Points Strategy

Character Limit
500 max

Recommended: 200-250 per bullet

Structure
3-Part

Feature + Benefit + Detail

Bullet Point Structure:
[CAPITALIZED FEATURE] - [Benefit] - [Supporting detail with keywords]

5-Bullet Keyword Strategy:

Bullet 1: Primary Keywords + Main Benefit

Bullet 2: Secondary Keywords + Unique Feature

Bullet 3: Long-tail Keywords + Use Case

Bullet 4: Material/Quality Keywords + Specifications

Bullet 5: Remaining Keywords + Warranty/Guarantee

üéØ CRITICAL:

EVERY bullet must include multiple relevant keywords NATURALLY - NO keyword stuffing!

  • Include high-CPI keywords
  • Use plural AND singular variations
  • Example: "breastfeeding pillows" vs "breastfeeding pillow"

Backend Search Terms (Hidden Keywords)

‚ö†Ô∏è MAXIMUM: 249 BYTES (Updated 2026)

These keywords are hidden from customers but Amazon indexes them for search

‚úì DO Include:

  • Synonyms ("lamp" if you used "light")
  • Alternative product names
  • Common customer misspellings
  • Related use cases
  • Material variations
  • Abbreviations
  • Different language terms (Spanish, etc.)

‚úó NEVER Include:

  • Keywords in Title or Bullets (wastes space!)
  • Punctuation: commas, hyphens, periods
  • Brand names (yours or competitors)
  • ASINs
  • Temporary statements: "new", "on sale"
  • Both singular AND plural (pick one)
  • Misspellings of common words
Correct Format:
hanging lamp ceiling fixture adjustable vintage retro farmhouse

Just use SPACES - NO commas, NO punctuation!

Complete Implementation Workflow

Phase 1: Research (Week 1)

Identify Main Keyword

Find your primary keyword phrase (2-4 words)

  • Use Amazon autocomplete
  • Check Helium 10 Magnet
  • Verify decent search volume (1,000+)

Find 3-10 Top Competitors

  • Search main keyword on Amazon
  • Select organic (non-sponsored) top results
  • Use Xray to analyze revenue
  • Extract ASINs from URL

Run Cerebro Analysis

  • Input all competitor ASINs
  • Set filters: Position 1-50, CPR < 50
  • Export keyword list
  • Target 4,000+ keywords discovered

Clean Keywords with Frankenstein

  • Remove duplicates
  • Remove competitor brand names
  • Sort by search volume
  • Group by match type (broad/phrase/exact)

Prioritize Keywords

Priority Criteria Where to Use
üî¥ HIGH Search volume 1,000+, CPR < 30 Title, Bullet 1-2
üü† MEDIUM Search volume 500-1,000, CPR 30-50 Bullet 3-5, Description
